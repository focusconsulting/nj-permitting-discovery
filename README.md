# NJ DCA Permitting: Source Documents

Public source documents on New Jersey construction permitting: the governing statutes and regulations, DCA process guidance, the current e-permitting systems, comparable systems in other jurisdictions, and historical construction-activity data.

All files are public records from the NJ Department of Community Affairs, the state legislature, or published references. Full file index in [`CATALOG.md`](CATALOG.md).
